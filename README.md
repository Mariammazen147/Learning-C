Introduction
This project is a collection of C programming exercises intended to enhance your understanding of key C concepts through practical examples. Each folder in the repository focuses on specific concepts like string manipulation, functions, and arrays.

Exercises Overview
Here is a brief description of the key exercises included:

2D Array: Demonstrates multidimensional array usage.
Calculator: Implements a basic calculator with addition, subtraction, multiplication, and division functions.
Average of Array: Calculates the average of elements in an array.
Even or Odd Checker: Checks whether a given number is even or odd.
Matrix Sum: Adds two matrices.
Swap Using Pointers: Swaps two numbers using pointers.
String Manipulation: Basic string operations like concatenation, comparison, and length.
